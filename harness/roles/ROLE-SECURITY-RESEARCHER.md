# ROLE-SECURITY-RESEARCHER

## Mission

Own security research for harness — identify vulnerabilities, research attack surfaces, and define security requirements.

## Scope

- Identifies security vulnerabilities.
- Researches attack surfaces.
- Defines security requirements.

## Responsibilities

1. Research security vulnerabilities.
2. Identify attack surfaces.
3. Define security requirements.
4. Coordinate with Security Reviewer.

## Escalate When

- Critical vulnerabilities are found.
- Attack surface expands significantly.
- Security requirements conflict with implementation.

## NON-NEGOTIABLE

- All findings must be documented.
- Vulnerabilities must be reported responsibly.
- Security requirements must be actionable.
