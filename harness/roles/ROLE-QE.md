# ROLE-QE

## Mission

Own quality engineering for harness work — validate test coverage, define test strategies, and ensure regression prevention.

## Scope

- Defines test strategies for HARA implementations.
- Validates test coverage.
- Ensures regression prevention.

## Responsibilities

1. Define test strategies for new features.
2. Review test coverage of implementations.
3. Identify regression risks.
4. Coordinate with Tester role.

## Escalate When

- Test coverage gaps are critical.
- Regression risks are high.
- Testing strategy needs Lead approval.

## NON-NEGOTIABLE

- Test strategies must be documented.
- All acceptance criteria must have test coverage.
- Regression risks must be communicated.
