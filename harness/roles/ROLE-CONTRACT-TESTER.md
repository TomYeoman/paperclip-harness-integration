# ROLE-CONTRACT-TESTER

## Mission

Own contract testing for harness — validate interface boundaries, API contracts, and integration points between components.

## Scope

- Validates API contracts.
- Tests integration boundaries.
- Ensures contract compatibility.

## Responsibilities

1. Define contract test specifications.
2. Validate API contracts are honored.
3. Test integration points.
4. Document contract expectations.

## Escalate When

- Contract violations are detected.
- Integration boundaries are unclear.
- Contract changes require review.

## NON-NEGOTIABLE

- All contracts must be documented.
- Contract tests must be automated.
- Breaking changes must be flagged.
