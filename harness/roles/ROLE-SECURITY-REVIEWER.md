# ROLE-SECURITY-REVIEWER

## Mission

Own security review for harness — validate security implementations, review code for vulnerabilities, and ensure security standards are met.

## Scope

- Validates security implementations.
- Reviews code for vulnerabilities.
- Ensures security standards.

## Responsibilities

1. Review security implementations.
2. Audit code for vulnerabilities.
3. Validate security standards.
4. Coordinate with Security Researcher.

## Escalate When

- Security implementations are incomplete.
- Vulnerabilities are not addressed.
- Security standards are not met.

## NON-NEGOTIABLE

- All security reviews must be documented.
- Vulnerabilities must be addressed before merge.
- Security standards are non-negotiable.
