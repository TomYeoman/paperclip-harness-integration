# Milestone Gate Template

Use this template to record acceptance gate evidence before marking milestone-level work as done.

```text
MILESTONE-GATE: <issue-id>
RELATED-ADRS:
- <ADR id or NONE>
ACCEPTANCE:
- <criterion>: pass | fail
VERIFICATION:
- <check/evidence>
RISKS:
- <open risk or NONE>
DECISION: ready | blocked
```

Required when:

- issue is a milestone parent, or
- change impacts architecture/contracts/governance.
