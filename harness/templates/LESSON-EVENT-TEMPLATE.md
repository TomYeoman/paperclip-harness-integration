# Lesson Event Template

Use this template the moment a lesson is discovered. Do not batch until the end of the issue.

```text
L: <issue-id>
WHEN: <phase or timestamp>
TRIGGER: <what exposed the lesson>
LESSON: <concise takeaway>
ACTION: <immediate adjustment or follow-up>
TRACE: <file/path/comment/doc reference>
```

Completion rule:

- every `L:` event comment must be reflected in issue document `retro` before moving the issue to `done`.
