## Core Signals

The harness communicates through issue comments and status transitions.

Common comment blocks:

- `DISCOVERY:`
- `REVIEW:`
- `DONE:`
- `BLOCK:`

Queue lifecycle signals:

- `QUEUE:`
- `CONFIRMED-D:`

Learning and milestone signals:

- `L:`
- `MILESTONE-GATE:`

## Interpretation Rule

Signal format is normative only when it matches canonical templates. Human examples are explanatory.
