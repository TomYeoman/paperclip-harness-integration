## Learning Loop

Learning is event-driven, not batch-only retrospection.

Required sequence:

1. Capture each lesson immediately with an `L:` comment.
2. Record trace and action for the lesson event.
3. Roll up lesson events into issue `retro` document before closure.

## Enforcement

- Lead enforces lesson evidence before `done`.
- Missing `L:`/retro chain blocks close readiness.
